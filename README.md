# Digital Ambassador Project

This repository contains all the tasks completed as part of the Digital Ambassador Module.  
The main goal of this project is to improve digital literacy, professional communication, cybersecurity awareness, and collaboration skills among students.

The project includes infographic design, digital portfolio creation, coding platform exploration, professional email writing, and cybercrime awareness resources. Each task focuses on practical digital skills that are useful in academic and professional environments.

## Objective of the Project

The objective of this project is to:

- Promote digital literacy among students
- Build professional online presence
- Improve coding and collaboration skills
- Teach proper email etiquette
- Create awareness about cybercrime and online safety

This project helps students understand how to use digital platforms responsibly and prepare for academic and professional environments.
